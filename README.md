### I'm a **full-stack QA & automation engineer**. I own quality end to end: strategy, framework, pipeline, and the release gate.

| Area | What I do |
| --- | --- |
| 🎭 **Test automation** | Playwright, Cypress, Selenium, and Appium for mobile, plus API automation with Postman and REST Assured. Page Object Model, anti-flake patterns, Allure reporting - across TypeScript, JavaScript, Java, and Python (pytest, JUnit, TestNG, Jest). |
| 🤖 **AI / LLM evaluation** | Golden-set eval harnesses, rubric scoring, RAG and hallucination checks, red-teaming and jailbreak resistance on the Anthropic and OpenAI APIs (Promptfoo, DeepEval, LangSmith, Braintrust), wired into a CI gate that blocks bad output before it ships. |
| ⚙️ **CI/CD & cloud** | GitHub Actions, GitLab CI, and Jenkins pipelines, Docker and Kubernetes, parallel runs on self-hosted and cloud runners (AWS, GCP, Azure) - so release day is boring. |
| 🔍 **Manual & exploratory** | Session-based exploratory charters, risk analysis, UAT, cross-browser and cross-device coverage, accessibility checks, and clear repro-first bug reports with logs and Sentry triage. |
| 🧭 **Strategy & release management** | Risk-based test plans, test-case design with boundary and equivalence analysis, coverage decisions, regression and release strategy, defect triage, quality metrics and gates, and process that scales with the team. |
